# Overview / 소개

Scratcher.js는 웹 및 다양한 프레임워크(React, Vue, React Native)에서 사용할 수 있는 스크래치 카드(긁기) 컴포넌트 라이브러리입니다. 이벤트, 프로모션, 쿠폰, 게임 등 긁기 효과가 필요한 다양한 상황에서 쉽고 빠르게 긁기 UI를 구현할 수 있습니다.

## 목적 및 활용 사례

- **이벤트/프로모션**: 긁어서 당첨 결과를 확인하는 마케팅 이벤트
- **온라인 쿠폰**: 긁어서 할인 코드, 시리얼 넘버 등 노출
- **게임/보상 UI**: 게임 내 보상, 미니게임 등 긁기 인터랙션

Scratcher 를 활용해 서비스중인 것이 있다면 공유해주세요!

## 설계 철학 및 구조

- 모듈화된 구조로 각 프레임워크에 최적화된 컴포넌트 제공
- 커스터마이징 용이: 브러시, 커버/배경, 콜백 등 다양한 옵션
- TypeScript로 작성되어 타입 안정성 보장

## 장점 및 차별점

- 쉽고 직관적인 API, 빠른 통합
- 다양한 커스터마이징(이미지, 색상, 콜백 등)
- 퍼포먼스 최적화 및 경량화
- 오픈소스 및 활발한 유지보수

## 커뮤니티 및 기여 안내

- 누구나 자유롭게 사용 및 기여 가능 (MIT 라이선스)
- 버그/기능 요청은 GitHub 이슈로, 개선/기여는 PR로 환영합니다.

## 주요 기능 / Features

- 긁기(스크래치) 효과 지원 (마우스, 터치 모두 대응)
- 다양한 커스터마이징 옵션 (브러시 크기, 덮개/배경 이미지, 콜백 등)
- React, Vue, React Native 지원
- 간단한 API와 빠른 통합
- TypeScript 지원
